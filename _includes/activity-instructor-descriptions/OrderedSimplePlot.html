This applet allows the user to input a series of (x,y) ordered pairs and plot them either in the order they were entered or as unconnected data points. Parameters such as minimum and maximum x value, minimum and maximum y value, and x- and y-scale can be set by the user or determined by the applet. This activity would work well in groups of two to four for about thirty minutes if you use the exploration questions, depending on how many shapes you graph.
