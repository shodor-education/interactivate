--- audiences: - "Grades 3-5" - "Grades 6-8" description: "Gives an introduction to sets and elements." subjects: - "Discrete" - "Number and Operations" title: "Sets" topics: - "counting" - "element" - "sets" ---

Student 1: So, a set is like a group of numbers that are the same type of thing, or, I mean, they have something in common?

Mentor: That's right! Do you know what an element is, then?

Student 1: Well, it is one of the numbers in a set.

Mentor: That's very good. Can anyone think of an example of this?

Student 2: Is 5 an element of the set of odd numbers?

Mentor: Great example! Can someone name an element of the set of prime numbers?

Student 3: I think 13 is an element of the set of prime numbers.

Mentor: You learn very fast! Can anyone think of an element of the set of perfect squares? Round fruit? Flavors of ice cream? Members of this class?

Continue questioning students until they see what it means to have a set and to be an element of a set.
